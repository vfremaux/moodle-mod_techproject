setStylePath(path)

  • path - {string} relative path to styles file

set path to styles file, default is “codebase/dhtmlxgantt.css”; used in simple printing method printToWindow 

    ganttChartControl.setStylePath("codebase/dhtmlxgantt.css");
    ganttChart.create("div2");
