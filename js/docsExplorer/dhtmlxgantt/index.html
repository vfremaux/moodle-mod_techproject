dhtmlxGantt

Use our Gantt chart to illustrate a project schedule. You may show the dependency relationships between activities as lines between them. It can be used to show current schedule status using percent-complete shadings. Projects, tasks and activities are comfortably organized into a tree structure (left to the chart), which can be expanded/collapsed by a single mouse click. Mouseover pop-up shows extended activity properties.

Initialize your chart directly from server with XML (or optionally using JavaScript API). Then, in edit mode, you may create new or delete activities, move them along the time line, resize, modify their properties, and you will instantly see those changes. The component will automatically apply all constraints and dependencies while you editing. Finally, you save new chart back to the server (in XML form) with a single function call.

Main Features

  • Multibrowser/Multiplatform support
  • Full control with JavaScript
  • XML support
  • Mouse-move editing of charts
  • Built-in EST date calculations
  • Nested tasks and multi-projects
